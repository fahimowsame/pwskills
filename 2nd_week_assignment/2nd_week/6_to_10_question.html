Ordered List of HTML Tags

  1. a Tag - The a tag defines a hyperlink, which is used to link from one page to another.
  2. br Tag - The br tag inserts a single line break.
  3. del Tag- The del tag defines text that has been deleted from a document. Browsers will usually strike a line through deleted text.
  4. em Tag- The em tag is used to define emphasized text. The content inside is typically displayed in italic.
  5. h1 to h6Tag- The h1to h6 tags are used to define HTML headings.
